# 03-basese-node
Contiene el desarrollo de una app de multiplicar

```
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     numero base para generar la tabla de multiplicar
                                                            [número] [requerido]
  -l, --listar   activar/desactrivar la impresion de la tabla de multiplicar
                                                     [booleano] [defecto: false]
  -h, --hasta    indica hasta que numero se va a multiplicar
                                                       [número] [defecto: false]
```
